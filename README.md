# Nightlies

Always run with `npm i --force` before you `ng serve`. And yes, it's not just you, it takes extra time to download these packages.

The `package.json` uses the latest builds from Angular `main`, aka the nightlies. ["How to keep up with Angular v19 features and releases" - Brandon Roberts](https://youtu.be/q4jOfQ7TbRg?t=57). The video also goes over how you can manually grab code from the build artifact of a pull request and drop them into a project.

[Reference gist](https://gist.github.com/michael-small/afa19b7448a4db8ae0ffdd0e8a5d08e9) for some cotnext on nightly version strings
