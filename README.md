# Nightlies

Always run with `npm i --force` before you `ng serve`. And yes, it's not just you, it takes extra time to download these packages.

The `package.json` uses the latest builds from Angular `main`, aka the nightlies. ["How to keep up with Angular v19 features and releases" - Brandon Roberts](https://youtu.be/q4jOfQ7TbRg?t=57). You can no longer grab build artifacts from PRs like in the video, so nightlies are about as good as it gets short of building the source code yourself.
